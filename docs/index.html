How this works

GitHub Pages serves this dashboard. Your Windows PC or VPS runs the Python backend next to MetaTrader 5. The dashboard calls the backend API, and the backend talks to MT5.

The included strategy is just a replaceable SMA crossover example, not a promise that it works. The point is to give you the plumbing: dashboard → Python API → MT5.
